# Boniface Alexander — Web Resume (Vite + React + Tailwind)

Static site scaffold for Boniface Alexander. Built with Vite, React and TailwindCSS.
The repo includes a GitHub Actions workflow to build and publish to GitHub Pages on push to `main`.

## Local dev
1. `npm install`
2. `npm run dev`
3. Open `http://localhost:5173/`

## Build
`npm run build` — output in `dist/`.
